### 1.4 智能派单算法 (核心)
**优先级**: P1 | **预计工时**: 20h | **状态**: 待认领

**任务描述**:
- 实现派单算法 v1
- 派单记录存证
- 算法可解释性

**算法设计**:
```
派单评分 = 距离权重(40%) + 负载权重(30%) + 效率权重(20%) + 公平权重(10%)
```

**技术要求**:
- 实时计算 (<200ms)
- 可解释性
- 单元测试覆盖率 > 90%

**产出物**:
- 算法文档 (docs/algorithm/dispatch.md)
- 单元测试

**认领人**: _____

## 📝 认领流程

1. 在 GitHub Issues 中创建任务 Issue
2. 在 Issue 中认领 (comment: "I want to claim this task")
3. 创建分支开始开发
4. 完成后提交 PR
5. 代码 Review 后合并
